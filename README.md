# Neovim Config

Minimal Neovim configuration with file tree, terminal, and run current file.

## Setup

```bash
git clone https://github.com/Rani367/vim-config.git ~/.config/nvim
```

## Keymaps

- `<Space>e` - Toggle file tree
- `<Space>t` - Toggle floating terminal
- `<Space>r` - Run current file in bottom terminal
- `<Esc>` - Exit terminal mode
